# Password Generator using Python

This is a simple Python project that generates a random password using letters, numbers, and special characters.

## Features
- Generates secure random passwords
- Uses Python built-in libraries
- Easy and beginner friendly project

## Technologies Used
- Python
- Random Module
- String Module

## How it Works
The program selects random characters from letters, digits, and special symbols to create a secure password.
